
==================
Introducing Shaker
==================

.. image:: images/shaker.jpg
	:width: 300px
	:scale: 40%

Shaker is a set of tools for creating and serving static asset rollups for Mojito applications. Shaker is tightly integrated with Mojito and the Cocktails platform.
